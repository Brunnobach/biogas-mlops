🌱 Biogas MLOps

End-to-end MLOps pipeline for biogas production forecasting — experiment tracking, model registry, containerized API and CI/CD.

View on GitHub

What this project demonstrates

MLOps MLflow FastAPI Docker CI/CD scikit-learn

This project shows how to take a biogas production forecasting problem and turn it into a production-ready ML system, including reproducible training, model versioning, API inference and automated testing.

Architecture

Raw Data → Feature Engineering → Model Training → MLflow Registry → FastAPI API (Docker)

Quick Start

git clone https://github.com/Brunnobach/biogas-mlops.git
cd biogas-mlops
pip install -r requirements.txt
python src/models/train.py
mlflow ui
python src/api/app.py
