🌱 Biogas MLOps

End-to-end MLOps pipeline for biogas production forecasting — experiment tracking, model registry, containerized FastAPI inference and CI/CD.

View on GitHub Connect on LinkedIn

What this project demonstrates

MLOps MLflow FastAPI Docker CI/CD scikit-learn pytest

This project takes a real industrial problem — forecasting biogas production in anaerobic digesters — and implements a complete production ML system. It includes data generation, feature engineering, model training with MLflow, a FastAPI inference service and automated testing.

6+ Features
5K Training rows
R² > 0.85 Target performance

Architecture

Raw Data → Feature Engineering → Model Training → MLflow Registry → FastAPI Inference API
  • Data generation: Synthetic biogas operational data inspired by real biodigesters
  • Feature engineering: Time-based cyclical features, interaction terms and lag variables
  • Model: Gradient Boosting Regressor with scikit-learn pipeline
  • Tracking: MLflow for experiments, parameters, metrics and artifacts
  • Serving: Dockerized FastAPI with /predict and /health endpoints

Quick Start

git clone https://github.com/Brunnobach/biogas-mlops.git
cd biogas-mlops
pip install -r requirements.txt
python src/data/generate_dataset.py
python src/models/train.py
uvicorn src.api.app:app --reload

Predict

curl -X POST "http://localhost:8000/predict" \
  -H "Content-Type: application/json" \
  -d '{
    "temperature": 37.0, "ph": 7.2, "organic_load": 3.5,
    "hydraulic_retention_time": 25, "substrate_type": "cattle_swine_poultry",
    "humidity": 85, "ambient_temperature": 22,
    "previous_day_production": 120, "month": 6
  }'

Docker Compose (full stack)

docker-compose up -d

Access MLflow at http://localhost:5000 and API docs at http://localhost:8000/docs.
